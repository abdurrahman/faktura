# Faktura Web API

Faktura is a RESTful API that provides querying invoices and creates new invoices.

## Installation

```shell
git clone https://github.com/abdurrahman/faktura.git
cd faktura
dotnet build
```

## Built with

* C#, .NET 5.0
* Polly
* Swagger
